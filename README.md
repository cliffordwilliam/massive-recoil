# Massive Recoil

Massive Recoil is a 2D pixel art offline single-player game built with Godot.

Engine-related notes are documented in `./docs/godot/`.
This directory acts as an **iterative, ever-growing knowledge base** covering topics such as tiles, resources, workflows, and other Godot development patterns.

Reusable prompts used during development are stored in `./prompts/`.

The documentation evolves alongside the project. If something breaks or a better approach is discovered, the knowledge base is updated accordingly.
